# 6인 보스 DPS 계산기 | 시즌2

Windows용 exe 배포 기준 사용 가이드입니다.
앱 내부 도움말 흐름에 맞춰 정리했습니다.

## 사용자 실행 방법
1. Releases에서 최신 exe 파일을 다운로드합니다.
2. exe를 실행합니다.
3. 설정은 자동 저장되며, 다음 실행 시 그대로 불러옵니다.

## 입력 방법 (도움말 기준)
1. 무기 선택
각 플레이어의 주무기를 드롭다운에서 선택합니다.

2. 인장 입력
[레]인 / [신]인 개수와 인장 공업 수치를 입력합니다.

3. 보조무기 스택 선택
등급별 스택(X / 0~9)을 선택합니다.
X는 해당 등급 미보유를 뜻합니다.

팁:
무기/인장/공업/보조무기는 마우스 휠로도 값 조정이 가능합니다.

## 오버레이/단축키 (도움말 기준)
- 위에 고정: 타이틀바 제거 + 게임 위 오버레이
- 오버레이 상태에서 ESC로 즉시 해제 가능
- 헤더 왼쪽 드래그로 창 이동 가능

- 1~6: 해당 플레이어 카드 ON/OFF
- F6: 스타 포커스 해제 후 계산기 창 포커스
- -: 현재 딜량/트라이 정보 채팅 전송
- F7: 매크로 문구 채팅 전송
- F12: 창 숨김/다시 표시
- ESC: 오버레이(위에 고정) 해제

## 자동 업데이트
앱 실행 시 최신 버전을 확인하고, 새 버전이 있으면 안내창이 표시됩니다.

사용자가 확인하면 다음 과정을 자동으로 진행합니다.
1. 최신 exe 다운로드
2. 파일 검증(sha256)
3. 기존 exe 교체
4. 앱 재실행

자동 업데이트 기준 파일은 [latest.json](latest.json)입니다.
형식 예시는 [latest.json.example](latest.json.example)을 참고하세요.

## 문제 해결
- 게임이 관리자 권한으로 실행 중이면 계산기도 관리자 권한으로 실행하세요.
- 게임과 계산기 권한(관리자 실행 여부)이 다르면 단축키/포커스 동작이 불안정할 수 있습니다.

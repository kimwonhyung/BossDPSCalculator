# 🔥 6인 보스 DPS 계산기  |  시즌2

## 실행 방법

### 방법 1 – Python으로 직접 실행
```bash
pip install customtkinter keyboard pyautogui pystray pillow
python main.py
```

### 방법 2 – EXE 파일로 빌드 (Windows)
```
build_exe.bat 더블클릭
→ dist/보스DPS계산기.exe 생성
```

### 방법 3 – 빌드 (macOS / Linux)
```bash
bash build_exe.sh
→ dist/보스DPS계산기 생성
```

## 필요 환경
- Python 3.9 이상
- customtkinter 5.2+
- pyinstaller 6.0+ (빌드 시)

## 기능
- 6명 플레이어 무기 / 인장수 / 공업 / 보조무기 입력
- 플레이어별 DPS 실시간 계산 및 막대 그래프 표시
- 20개 보스 클리어 가능 여부 + 예상 클리어 시간 표시
- 설정 자동 저장 (홈 디렉토리 .dps_calc_save.json)
- 온보딩 화면 (처음 실행 시 자동 표시, 이후 도움말 버튼으로 접근)
- 전역 단축키 지원 (`F6`, `-`, `1~6`)

## 단축키
- `1~6`: 해당 플레이어 카드 ON/OFF
- `-`: 현재 딜량 + 트라이 정보 채팅 전송
- `F6`: 스타 포커스 해제 후 계산기 창 포커스
- `ESC`: 오버레이(위에 고정) 해제

## 자동 업데이트(권장)
한 번 설치한 뒤에는 사용자가 새 EXE를 다시 받지 않도록 자동 업데이트를 사용할 수 있습니다.

### 1) 코드에서 버전/매니페스트 URL 설정
`main.py`의 아래 상수를 수정하세요.

```python
APP_VERSION = "1.32.0"
UPDATE_MANIFEST_URL = "https://<배포주소>/latest.json"
```

### 2) 서버에 latest.json 업로드
예시는 `latest.json.example` 파일을 참고하세요.

필드 설명:
- `version`: 최신 버전 문자열
- `url`: 최신 EXE 다운로드 URL
- `sha256`: 다운로드 파일 SHA256 (권장)
- `notes`: 업데이트 안내 문구(선택)

### 3) 새 버전 배포 시
1. 새 EXE 빌드
2. EXE 업로드
3. EXE SHA256 계산
4. `latest.json`의 `version/url/sha256` 갱신

앱은 실행 시 최신 버전을 확인하고, 사용자가 확인하면 자동으로 다운로드/교체 후 재실행됩니다.

## 게임 중 사용 팁
- 스타크래프트 창모드에서 `F6`을 누르면 게임 입력 캡처를 끊고 계산기로 포커스를 가져옵니다.
- 게임이 관리자 권한으로 실행 중이면 계산기도 관리자 권한으로 실행해야 단축키/포커스 전환이 안정적입니다.
- 게임/계산기 권한(관리자 실행 여부)이 다르면 포커스 전환 안정성이 떨어질 수 있습니다.

## 계산 공식
```
주무기 DPS  = (무기기본값 + 인장수 × 인터셉터DPS) × 180
보조무기 DPS = 등급기본값 × (1 + 스택 × 0.5) × 180
총 딜량     = 6명 DPS 합산
클리어 가능 = 총 딜량 ≥ 보스 체력
```
